// THIS DATABASE CONTAINS ALL SKILL DATA INCLUDING TALENTS BUT NOT TALENT GRAPH (POSITIONAL) INFORMATION
// TALENT GRAPH IS CURRENTLY UNMODIFIABLE

// LINE FORMAT IS AS FOLLOWS:
// name|tooltip|icon file name|level requirement|AE cost|TE cost|class|spec|max talent ranks|
// the line is split by the pipe '|' character. these must be placed in the appropriate places or the line will not load correctly.
// the line MUST end with a closing pipe | after the max talent ranks or the line will not load correctly.

// LINE FORMAT DETAILS:
// name (string)
//	the name of the skill. this must match exactly with talent graph data if it is a talent to be shown on the graph
// tooltip (string)
//	the complete tooltip shown when mouse over. accepts text escape sequences such as \t for tabs and \n for new lines
// icon file name (string)
//	the name of the file for the skill's icon as seen on wowhead
//	some icons may not be included in the application and will not work (like wrath stuff)
//	does not include the file extension. only the name
// level requirement (integer)
//	a number between 1 and 60
// AE cost (integer)
//	the AE cost of this skill
// TE cost (integer)
//	the TE cost of this skill. should be 0 for non-talents
// class (string)
//	the exact case-sensitive match of the class this skill belongs to
//	Druid Hunter Mage Paladin Priest Rogue Shaman Warlock Warrior
// spec (string)
//	the exact case-sensitive match of the class's specialization this skill belongs to
//	Arms Holy Shadow Feral etc
//	use "BeastMastery" for BM hunters
//	redundant spec names such as Holy and Restoration do not require any extra identifiers
// max talent ranks (integer)
//	the max ranks, usually 1->5, if this is a talent.
//	use 0 for non-talent skills
//	this number is used internally to distinguish between talents and abilities
